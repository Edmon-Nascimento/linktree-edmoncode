# Linktree – Personal Links Page

This project is a simple and responsive Linktree-style webpage built with pure HTML and CSS. It centralizes personal and professional links in one place, making it easy to share contact information, social profiles, and portfolio access.

## Overview

The page presents a personal profile with a photo, short bio, availability status, and a list of external links such as LinkedIn, GitHub, Instagram, portfolio, CV download, WhatsApp, and email.

It is designed to be lightweight, fast, and easy to deploy on any static hosting service.

## Features

- Responsive layout
- Clean and minimal design
- Profile image and bio section
- Centralized social and professional links
- Downloadable CV
- External links opening in a new tab
- Custom favicon
- Google Fonts integration

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
